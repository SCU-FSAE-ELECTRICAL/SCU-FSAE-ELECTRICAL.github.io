# SCU FSAE Electrical Team Website

Official GitHub Pages website for the Santa Clara University Formula SAE Electrical Team - Bronco Racing.

## About

This is a Jekyll-based website showcasing our team's projects, achievements, and activities in Formula SAE Electric competition.

## Development

### Prerequisites

- Ruby 3.0 or higher
- Bundler gem
- Jekyll gem

### Local Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/SCU-FSAE-ELECTRICAL/SCU-FSAE-ELECTRICAL.github.io.git
   cd SCU-FSAE-ELECTRICAL.github.io
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Run the local development server:
   ```bash
   bundle exec jekyll serve
   ```

4. Open your browser to `http://localhost:4000`

### Making Changes

- Pages are in the root directory as `.md` files
- Blog posts go in the `_posts` directory with the format `YYYY-MM-DD-title.md`
- Layouts are in `_layouts`
- Includes (reusable components) are in `_includes`
- CSS styles are in `assets/css/style.scss`
- Site configuration is in `_config.yml`

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
